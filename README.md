# ПК «Выпарная батарея» - Программный комплекс расчёта батарей выпарных установок

## Описание

Программный комплекс для автоматизации расчёта многокорпусных и вакуумных выпарных установок с анализом многокорпусных систем, вторичного пара и вакуумных условий.

## Возможности

### Основные функции
- ✅ Расчёт технологических параметров многокорпусных и вакуумных выпарных установок
- ✅ Анализ эффективности батарей выпарных аппаратов
- ✅ Оптимизация числа корпусов и расхода пара
- ✅ Формирование детализированных отчётов (PDF, Excel)
- ✅ Графическая визуализация результатов (графики температуры, давления, расхода пара)
- ✅ Материальный и тепловой баланс каждого корпуса
- ✅ Подбор площади теплообмена, расхода пара, распределения температур и давления

### Типы поддерживаемых установок
- Прямоточные многокорпусные
- С принудительной циркуляцией
- Плёночные (восходящая и падающая плёнка)
- Вакуумные

## Установка и запуск

### Требования
- Node.js >= 16
- Python 3.12+ (для разработки)
- Docker и Docker Compose (для продакшена)

### Запуск в режиме разработки

1. Клонировать репозиторий:
```bash
git clone <repository-url>
cd himia-11klass
```

2. Установить зависимости фронтенда:
```bash
npm install
```

3. Установить зависимости сервера:
```bash
cd server
npm install
cd ..
```

4. Настроить переменные окружения:
```bash
cp server/.env.example server/.env
# Отредактировать .env файл
```

5. Запустить сервер:
```bash
cd server
npm start
```

6. В другом терминале запустить фронтенд:
```bash
npm start
```

7. Открыть браузер: `http://localhost:3001`

### Запуск через Docker

```bash
docker-compose up --build
```

Открыть: `http://localhost:3001`

## Использование

### Расчёт выпарной батареи

1. Перейти на страницу "Калькулятор"
2. Заполнить входные параметры:
   - Тип установки
   - Направление потока
   - Количество корпусов (1-10)
   - Расход исходного раствора
   - Начальная и конечная концентрация
   - Температура греющего пара
   - Коэффициент теплопередачи
   - Теплота парообразования и конденсации
3. Нажать "Рассчитать"
4. Просмотреть результаты:
   - Схема батареи
   - Графики (температура, давление, расход пара, площадь теплообмена)
   - Таблица результатов по корпусам
   - Сводные данные
5. Сохранить проект
6. Экспортировать в PDF или Excel

### Оптимизация

Нажмите кнопку "Оптимальное количество корпусов" для автоматического подбора оптимального числа корпусов по критерию максимальной паровой экономичности.

## Математическая модель

### Материальный баланс
Для каждого корпуса i количество испарённого растворителя:
```
W_i = G_i * (1 - C_i / C_{i+1})
```

Конечный расход раствора после n корпусов:
```
G_k = G_n - W_total
```

### Тепловой баланс
Тепловая нагрузка корпуса:
```
Q_i = W_i * r_i
```
где r - теплота парообразования растворителя

### Расход пара
Расход греющего пара на корпус:
```
D_i = Q_i * 3600 / R_i
```
где R - теплота конденсации греющего пара

### Площадь теплообмена
```
F_i = Q_i * 1000 / (K * ΔT)
```
где:
- K - коэффициент теплопередачи, Вт/(м²·К)
- ΔT - средний температурный напор, К

### Паровая экономичность
```
Steam Economy = W_total / D_total
```

## API Endpoints

### Авторизация
- `POST /api/auth/register` - Регистрация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Проверка токена

### Расчёты
- `POST /api/calculate` - Выполнить расчёт
- `POST /api/calculate/optimize` - Оптимизация количества корпусов
- `POST /api/calculations` - Сохранить проект
- `GET /api/calculations` - Получить все проекты
- `GET /api/calculations/:id` - Получить проект по ID
- `DELETE /api/calculations/:id` - Удалить проект
- `GET /api/calculations/:id/export/pdf` - Экспорт в PDF
- `GET /api/calculations/:id/export/excel` - Экспорт в Excel
